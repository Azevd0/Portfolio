Sobre mim

Meu nome é Davyson Arthur e sou Desenvolvedor Back-end focado em criar soluções sólidas com Java e Spring Boot. Acredito que um bom software nasce da união entre código limpo e infraestrutura inteligente — por isso, busco ir além da simples codificação, integrando Banco de Dados relacionais e Cache para garantir a performance e escalabilidade que sistemas modernos exigem. Minha experiência conta também com conteinerização de microsserviços via Docker e Docker Compose, que me permite entregar ambientes consistentes. Aplico diariamente os princípios de Clean Code e as melhores práticas de design de software para garantir que cada API construída seja funcional e de fácil manutenção. Atualmente, foco na construção de arquiteturas seguras e de alta disponibilidade, sempre buscando transformar desafios técnicos em soluções eficientes que agreguem valor real ao negócio e à experiência do usuário

Meus Principais Projetos

🍽️ Order Factory

Aplicação robusta e escalável, para gerenciamento de restaurante. O sistema permite registro de usuários e autenticação, possuindo restrição de acesso para administradores. O sistema permite lançar pedidos, adicionar comidas à mesas, registrar pagamento e consultar histórico de lançamento e financeiro da API. Para evitar sobrecargas ao banco de dados, a aplicação conta com cache do Redis executado em segundo plano via Docker Compose, tornando consultas mais rápidas e eficientes.

  • Java 17
  • Spring Boot
  • Spring Security
  • H2 Database
  • PostgreSQL
  • Springdoc
  • Redis
  • Docker Compose

💵 Gestor de Finanças

Solução completa para gestão de finanças pessoais ou empresariais. A aplicação é executada via terminal e permite registrar receitas e despesas, listar finanças por mês e categoria e gerar relatórios financeiros, exibindo para o usuário o quanto teve de lucro mensal.

A aplicação conta com duas versões: uma para desktop e outra levemente ajustada para dispositivos móveis que possuam um terminal (o Termux por exemplo).

  • JPA
  • Hibernate
  • Maven
  • PostgreSQL
